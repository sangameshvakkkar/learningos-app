# LearningOS Release v0.1

Initial production-quality starter release.

## Included

- React, Vite, TypeScript, Tailwind CSS frontend.
- React Router, Axios, React Query, React Hook Form, and Zod.
- Landing, login, register, dashboard, course catalog, profile, sidebar, navbar, and dark mode.
- FastAPI backend with JWT authentication, users, courses, enrollments, health endpoint, and Swagger.
- SQLAlchemy 2 models and Alembic initial migration.
- PostgreSQL via Docker Compose.
- Dockerfiles, environment examples, database reference schema, and documentation.

## Not included

- Kubernetes, Terraform, AWS, or CI/CD.
